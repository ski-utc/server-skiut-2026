Politique de Confidentialité

1. Introduction

La présente politique de confidentialité décrit comment nous collectons, utilisons, partageons et protégeons les données personnelles des utilisateurs de notre application. En utilisant cette application, vous consentez à la collecte et à l'utilisation de vos informations conformément à cette politique.

3. Types de Données Collectées

Nous collectons les types de données suivantes, en fonction des fonctionnalités de l'application :

  • Données personnelles : Nom, prénom, adresse e-mail.
  • Données de localisation : Si vous avez activé la géolocalisation, nous collectons des informations liées à votre localisation.
  • Données techniques : Identifiant de l'appareil, adresse IP.

4. Utilisation des Données

Les données collectées sont utilisées pour les finalités suivantes :

  • Fournir les services de l'application : Création et gestion de votre compte, personnalisation de l’expérience utilisateur, fonctionnalités.
  • Communication : Envoi de notifications, mises à jour de l'application, ou messages marketing avec votre consentement.
  • Conformité légale : Respect des obligations légales et réglementaires (facturation, sécurité, etc.).

5. Partage des Données

Nous partageons vos données personnelles avec les entités suivantes :

  • Nos prestataires de services : Hébergement, paiement, support technique, etc.
  • Les autorités compétentes : Si requis par la loi ou en cas de demande légale.

6. Sécurité des Données

Nous prenons des mesures de sécurité appropriées pour protéger vos données personnelles contre toute utilisation abusive, perte, ou altération. Cela inclut l'utilisation du chiffrement (SSL/TLS) pour protéger les données sensibles pendant la transmission.

7. Rétention et Suppression des Données

Nous conserverons vos données personnelles aussi longtemps que nécessaire pour remplir les finalités pour lesquelles elles ont été collectées et pour respecter nos obligations légales. Vous pouvez demander la suppression de vos données personnelles à tout moment en nous contactant.

8. Droits des Utilisateurs

Conformément au RGPD, vous disposez des droits suivants :

  • Droit d’accès : Vous avez le droit de demander l’accès aux données personnelles que nous traitons à votre sujet.
  • Droit de rectification : Vous pouvez demander la correction de toute donnée inexacte ou incomplète.
  • Droit d’effacement : Vous pouvez demander la suppression de vos données personnelles sous certaines conditions.
  • Droit d’opposition : Vous pouvez vous opposer au traitement de vos données pour des raisons légitimes.
  • Droit à la portabilité : Vous pouvez obtenir une copie de vos données dans un format structuré, couramment utilisé et lisible par machine.

Pour exercer vos droits, veuillez nous contacter à l'adresse suivante : asso.skiutc@gmail.com.
